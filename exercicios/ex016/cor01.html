Cores em CSS

Formas de representar cores

As cores podem ser representadas de várias formas no CSS, como por exemplo:

  • Por nome (red, blue, green)
  • Por valores hexadecimais (#ff0000 para vermelho)
  • Por valores RGB (rgb(255, 0, 0) para vermelho)
  • Por valores RGBA (rgba(255, 0, 0, 0.5) para vermelho com opacidade)
  • Por valores HSL (hsl(0, 100%, 50%) para vermelho)
  • Por valores HSLA (hsla(0, 100%, 50%, 0.5) para vermelho com opacidade)
    •

Essas diferentes formas de representar cores permitem flexibilidade e controle sobre a aparência dos elementos em uma página web.

Você pode usar essas representações de cores para estilizar elementos HTML, como textos, fundos e bordas.
