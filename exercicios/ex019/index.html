Criando Sites com HTML e CSS

Aprendendo HTML

HTML básico

Aprenda sobre as tags fundamentais como <p>, <h1>, e <a> para estruturar suas primeiras páginas da web com conteúdo semântico.

HTML intermediário

Explore formulários, tabelas e a incorporação de multimídia como áudio e vídeo para criar páginas mais interativas e ricas em conteúdo.

HTML avançado

Descubra recursos avançados como a API de Canvas para gráficos, Web Storage para armazenamento de dados no navegador e Web Components para criar elementos reutilizáveis.

Aprendendo CSS

CSS básico

Entenda como usar seletores, propriedades de cor, fonte e espaçamento para estilizar seus documentos HTML e dar vida ao seu design.

CSS intermediário

Aprofunde-se em Flexbox e Grid para criar layouts complexos e responsivos. Aprenda sobre pseudo-classes e pseudo-elementos para estilizações mais dinâmicas.

CSS avançado

Domine técnicas como animações, transições e variáveis CSS para criar interfaces de usuário modernas e interativas. Explore pré-processadores como SASS para otimizar seu fluxo de trabalho.
